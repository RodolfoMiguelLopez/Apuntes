# 📚 Apuntes Técnicos

Repositorio personal de apuntes, guías y configuraciones técnicas sobre redes, Linux, DevOps, seguridad, infraestructura y más.

> **Estructura:** Los apuntes están organizados por temática en carpetas. Cada archivo puede ser una guía completa, apuntes de curso o paso a paso de configuración.

---

## 📁 Estructura del Repositorio

| Carpeta | Descripción |
|---|---|
| `redes/` | Cisco, protocolos de red, VLANs, MTA 98-366, backup de switches |
| `linux/` | Guías de Linux, comandos, permisos, NDG Linux Essentials |
| `devops/` | Docker, Git, Python |
| `seguridad/` | Checkpoint, Metasploit, SSH |
| `infraestructura/` | Netbox, HP DataProtector |
| `herramientas/` | Markdown, Sublime Text, Mac |
