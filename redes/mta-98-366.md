## Conceptos básicos

Término | Significado
---|---
Broadcasting | Información enviada a cada host en la red; solo el receptor de destino la conserva, el resto la descarta.
Unicast | La información se envía a un solo host.
Multicast | *(pendiente de completar)*
Tasa de transferencia / tasa de bits | Máximo de bps que pueden ser transmitidos por la red.
Hub | Dispositivo básico de conexión. Conecta hosts por cables de cobre.
Switch | Conecta múltiples tramos de red en una sola red; solo retransmite la información hacia los tramos con destinatario.
Router / SOHO (Small Office, Home Office) | Dispositivo central de conexión con enlace especial a internet.
Adaptador de red (NIC) | Tarjeta de interfaz de red. Permite enviar y recibir información. Tiene su propia CPU y ROM.
RJ45 / 8P8C | Interfaz física utilizada para conectar redes con cableado estructurado. [+info](https://es.wikipedia.org/wiki/RJ-45)
Full Duplex | El adaptador puede enviar y recibir información simultáneamente.
Half Duplex | El adaptador envía y recibe información, pero no al mismo tiempo.

Cuando la información es transferida en una LAN, se envía de forma **serial** (1 sola cadena de bits a la vez).

## Dirección IP

Permite a cada computadora enviar y recibir información de forma ordenada y eficiente.

Cada dirección IP se divide en 2 partes: `192.168.1.1`

- La porción de red: `192.168.1`
- La porción de host: `1`

Para saber cuál es el host se usa la máscara de subred:

- Todos los `255` → porción de red
- Todos los `0` → porción de host

```
192.168.0.1
255.255.255.0
```

> **Cada host en una red necesita una IP diferente para evitar conflictos.**

Las direcciones IP son números de punto decimal de 32 bits. Cada dirección contiene 4 números (bytes / octetos):

| Tipo | Equivalente |
|---|---|
| Decimal con puntos | 201.161.1.226 |
| Hexadecimal con puntos | 0xC9.0xA1.0x01.0xE2 |
| Octal con puntos | 0311.0241.0001.0342 |
| Binario con puntos | 11001001.10100001.00000001.11100010 |

Comandos útiles:

| Comando | Función |
|---|---|
| `ipconfig` | Ver la configuración de red |
| `ping` | Comprobar la conectividad de la red |
